# Cabinet Dentaire – Charlotte *****

Site web vitrine professionnel pour un cabinet dentaire, conçu pour être responsive, clair et prêt à l’emploi.

## Fonctionnalités
- Présentation du cabinet
- Informations pratiques (horaires, contact, localisation)
- Formulaire de contact
- Design professionnel et rassurant
- Structure évolutive

## Objectif
Offrir une présence en ligne claire et professionnelle adaptée aux besoins d’un cabinet médical.
